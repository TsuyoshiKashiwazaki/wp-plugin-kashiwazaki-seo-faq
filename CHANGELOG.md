# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-21

### Added
- 初回リリース
- ブロックエディタ対応（投稿、固定ページ、カスタム投稿で使用可能）
- アコーディオン型表示（クリックで開閉）
- シンプル型表示（常に表示）
- FAQPage構造化データ（JSON-LD）対応
- カスタマイズ可能なデザイン設定
- 質問・回答アイコンの選択機能（各6種類）
- 1,677万色カラーピッカー
- 文字サイズ・アイコンサイズの設定
- リアルタイムプレビュー機能
- タブ式設定画面（基本設定/デザイン設定/使い方）
